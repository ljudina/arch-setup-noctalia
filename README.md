# Arch Linux Development Environment Setup (Noctalia)

This repository contains a modular, reproducible Arch Linux setup for a modern Wayland-based development environment using Hyprland, Noctalia Shell (panel, launcher, lock screen, notifications, wallpaper theming), Oh My Zsh with Powerlevel10k, Neovim (php.nvim), Ghostty, and a symlink-based dotfiles workflow.

The main goals are:
- fast setup on a new machine
- zero configuration drift
- easy maintenance and updates
- no duplicated configuration files

---

## Features

- Modular `install.sh` with ordered setup steps
- Pacman and AUR package management
- Hyprland desktop configuration
- Noctalia Shell (replaces Waybar, Rofi, Hyprlock, Hyprpaper)
- Ghostty terminal configuration
- Oh My Zsh with Powerlevel10k
- Arch-native Zsh plugins (autosuggestions and syntax highlighting)
- Neovim PHP-focused config via upstream repo (`ljudina/php.nvim`)
- Symlink-based dotfiles (single source of truth)
- Safe to re-run at any time

## Installation

### 1. Clone the repository

```bash
git clone <your-repo-url> ~/src/arch-setup-noctalia
cd ~/src/arch-setup-noctalia
```

### 2. Run the installer (do NOT use sudo)

```bash
chmod +x install.sh
./install.sh
```

The installer will:
- install pacman packages
- install AUR packages using paru (including Noctalia Shell)
- install Oh My Zsh and Powerlevel10k
- run Noctalia initial setup
- install the Neovim php.nvim configuration
- enable required services
- create symlinks for dotfiles in `~` and `~/.config`

### 3. Open a new terminal

Or log out and back in if Zsh was set as the default shell.

---

## Re-running and Updating

This setup is idempotent and safe to re-run.

```bash
./install.sh
```

To update everything:

```bash
git pull
./install.sh
```

## Desktop Shell: Noctalia

Noctalia is a Wayland desktop shell built on Quickshell (Qt/QML). It replaces Waybar, Rofi, Hyprlock, and Hyprpaper with a unified shell experience.

Noctalia provides:
- Status bar / panels
- Application launcher
- Lock screen
- Notification system with history and DND
- OSD for volume and brightness
- Dock
- Wallpaper management
- GTK/Qt theming via Matugen (Material Design 3)

Hyprland keybinds invoke Noctalia via `noctalia ipc call <module> <action>`.

---

## Dotfiles Workflow

All configuration files are managed using symlinks.

- Editing `~/.config` or `~/.zshrc` edits the repository directly
- No configuration files are generated by scripts
- No duplication between scripts and dotfiles

---

## Neovim Configuration

Neovim uses the upstream configuration:

https://github.com/ljudina/php.nvim

The installer:
- clones the repo into `~/.local/share/nvim-php-config`
- symlinks it to `~/.config/nvim`

To update Neovim config:

```bash
git -C ~/.local/share/nvim-php-config pull
```

## Zsh Setup Notes

- Oh My Zsh is installed automatically
- Powerlevel10k is the active theme
- Zsh plugins are installed via pacman
- `zsh-autosuggestions` and `zsh-syntax-highlighting` are sourced explicitly
- `navi` is integrated via `eval "$(navi widget zsh)"`
